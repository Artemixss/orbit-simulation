Planetary Simulation Engine
N-Body gravity simulation using VPython

Features

Includes each planet in solar system with their actual masses
Forces are calculated with Newton's Law of Universal Gravitation and Newton's Third Law of Motion.
This engine uses Velocity Verlet algorithm.

Prerequisites

Python 3.x and the VPython library.
```bash
   pip install vpython
   python main.py
```

Controls

Right-Click + Drag: Rotate the 3D camera.
Scroll Wheel / Alt + Drag: Zoom in and out.
Shift + Left-Click + Drag: Pan the camera.
Camera Focus Menu
Simulation Speed Slider
